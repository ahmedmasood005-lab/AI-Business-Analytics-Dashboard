# AI Business Analytics Dashboard

The AI Business Analytics Dashboard is a web-based business intelligence application developed using Python, Streamlit, SQLite, and Machine Learning. It enables users to analyze sales data through interactive dashboards, monitor key performance indicators (KPIs), visualize business performance with dynamic charts, and predict future sales revenue using a trained machine learning model. The application includes a secure authentication system with user registration, login, and password reset functionality, ensuring safe access to business data.

The dashboard provides powerful analytics features, including revenue analysis, sales trends, category and region filtering, KPI cards, and AI-based sales prediction. Users can also generate and download reports in CSV, Excel, and PDF formats, making it easier to share insights and support business decision-making. The project is built using Python, Streamlit, Pandas, Plotly Express, Scikit-learn, Joblib, SQLite, OpenPyXL, FPDF, and Bcrypt, offering a complete and user-friendly business analytics solution with a modern interface and responsive design.

## Features

- Secure User Authentication (Login, Registration, Forgot Password)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/AI-Business-Analytics.git
   ```

2. Navigate to the project folder:
   ```bash
   cd AI-Business-Analytics
   ```

3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:
   ```bash
   streamlit run app.py
   ```

## Usage

Register a new account or log in with an existing account to access the dashboard. Explore sales analytics through interactive charts and KPI cards, use the AI prediction module to estimate future sales revenue, apply filters to analyze specific business segments, and download reports in CSV, Excel, or PDF format.
